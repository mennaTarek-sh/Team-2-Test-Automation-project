__________________________________________________________________________________________________________________________________________________________________________________________
 Testing Types Used

Manual Testing
Used to validate UI behavior, user flows, error messages, and usability.
Test cases are documented in Google Sheets with steps, expected results, and actual results.

Automation Testing (Selenium)
Used for critical and repetitive flows such as Login, Logout, and Checkout.
Automation helps save time and ensures fast regression testing after each update.

API Testing
Used to test backend logic and data flow independently from the UI.
Includes Authentication (JWT), CRUD operations, filters, and status code validation.
